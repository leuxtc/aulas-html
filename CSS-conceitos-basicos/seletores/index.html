Seletores CSS

Bem-vindo!

No CSS nós possuímos os seletores por ID

  • Seletor por tipo:Busca elementos por uma tag HTML específica
  • Seletor por ID(#):Busca elementos atráves do atributo "id"
  • Seletor por Classe(.):Busca elementos atráves do atributo "class"
  • Seletor por Universal(*):Seleciona todos os elementos HTML
  • Seletor por Atributo([atrib]):Seleciona elementos que possuem um atributo específico em nosso documento HTML e conseguimos também buscar atributos com um valor específico
